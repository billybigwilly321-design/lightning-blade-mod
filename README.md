# Lightning Blade Mod

A Minecraft 1.20.1 NeoForge mod that adds the **Lightning Blade** - a powerful sword with the ability to teleport!

## Features

- ⚡ **Lightning Blade**: A diamond-tier sword
- 🔄 **Teleport Ability**: Press **R** while holding the sword to teleport 30 blocks in the direction you're looking
- ⚡ **Lightning Effects**: Lightning spawns at your teleport destination
- 🎵 **Thunder Sound**: Thunder sound plays with each teleport
- 🛠️ **Easy Crafting**: Craft with wood logs

## Crafting Recipe

```
 W
 W
 S
```

Where `W` = Oak Log and `S` = Oak Log

## Controls

- **Hold the Lightning Blade** in your main or off-hand
- **Press R** to teleport 30 blocks forward with lightning effect
- **No cooldown** - unlimited teleports!

## Installation

1. Download the latest release
2. Place the `.jar` file in your `mods` folder
3. Launch Minecraft with NeoForge 1.20.1

## Building from Source

```bash
git clone https://github.com/billybigwilly321-design/lightning-blade-mod
cd lightning-blade-mod
./gradlew build
```

The compiled mod will be in `build/libs/`

## Requirements

- Minecraft 1.20.1
- NeoForge 47.2.0 or later
- Java 17 or later
